Overview
This utility helps you edit assets in bulk by using comma separated values (CSV).

Work flow

  1. Select your asset type
  2. Select the properties that you wish to edit
  3. Paste the data into the Input field
  4. Analyze or Submit

Gotcha's

  • Groups: Groups must not have trailing or leading spaces and they must be separated by ;
    Also important to note is the system name for the default Entire Organization group, "GroupCompanyId"
  • NFC Keys: MyGeotab does not allow duplicate keys in the database. If re-using you must clear keys prior to assigning them to a new driver. You can do this in bulk by passing through "CLEARKEY" as the NFC Key ID
  • Expire Zone/Device/User: Expected value is a boolean, either TRUE or FALSE. TRUE will make the entity HISTORICAL, FALSE will activate the entity(will not work for Devices as SN is remvoed)
  • Security Levels: The system ID's, which need to be used, for the default security levels are:
    • Administrator: GroupEverythingSecurityId
    • Default user: GroupUserSecurityId
    • Supervisor: GroupSupervisorSecurityId
    • View Only: GroupViewOnlySecurityId
    • Nothing: GroupNothingSecurityId
  • Time Zone: MyGeotab uses Olson Time Zones, you can see a list of them at this link
  • Work Schedule: The system names, which need to be used, for the default work schedules are:
    • Standard Hours: WorkTimeStandardHoursId
    • Lunch Hours: WorkTimeLunchHoursId
    • Early Leave Hours: WorkTimeEarlyLeaveHoursId
    • Late Arrival Hours: WorkTimeLateArrivalHoursId
    • All Hours: WorkTimeAllHoursId
